# Blog

![image-20210726090159966](https://gitee.com/steam1211211/my-pic/raw/master/image/image-20210726090159966.png)

#### 软件架构
- 数据库：MySQL8.0.xx
- spring boot 2.3.4
- Mybatis Plus 3.4.1
- spring-cloud-starter-alibaba-nacos-config-2.2.3.RELEASE.jar
